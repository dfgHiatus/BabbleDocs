# Notes on MacOS

Desktop Notifications do not work on MacOS. This does not impact other parts of the Babble App.

## Building from source

If building from source, you will need to remove the line `onnxruntime-directml==X.XX.X` from the `requirments.txt` file inside the `BabbleApp` folder.
